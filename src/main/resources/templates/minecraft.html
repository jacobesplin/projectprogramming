How to build a minecraft sever

Software:

Download the latest server (server 1.17)-Current as of this writing. Make sure you are using java version OpenJDK 16.

How To:

Setting up the server is relatively easy. All you need to do is run the server and it does all the work.

  • Open a terminal or command prompt
  • Type java -jar server.jar

And Enjoy!

Issues? Please see our troubleshooting guide

Sunset

Somethings you may want to know before launching your server!

  • Running the server on the same computer you are gaming on may cause memory issues and slow down the server.
  • There maybe some security risks.

Trouble Shooting
Issues starting server
If you are having issues getting the server to start because of a java runtime issue make sure you are using the correct java version. Check this by typing java --version in the command prompt/terminal.
